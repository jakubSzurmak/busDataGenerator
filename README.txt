In terms of database implementation, the first step is opening SQL Server Management Studio or other software
dedicated for controlling your SQL server instance and connect to it. Next steps will be presented for Microsoft SQL Server Management Studio.
Press New Query and paste the contents of the file "SQL.tex" into the newly created window.
Now scroll down untill you see the BULK INSERT querries, in the place of YOURDIRECTORY fill in the directory
in which the files created by the generator are for example: "C:\Users\Jakub\PycharmProjects\DwGen" without
quotes. Last thing to do is to run the script.
